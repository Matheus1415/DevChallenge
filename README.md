# DevChallenge

DevChallenge é um repositório de prompts especializados para agentes de IA focados em carreira, recrutamento, aprendizado e desenvolvimento de software.

O projeto reúne agentes capazes de analisar vagas, identificar competências exigidas pelo mercado, recomendar oportunidades, gerar desafios técnicos personalizados, avaliar perfis profissionais e orientar desenvolvedores em sua evolução técnica.

Os prompts foram projetados para serem utilizados em fluxos de automação, agentes autônomos e plataformas de IA, permitindo criar experiências inteligentes voltadas para empregabilidade, capacitação e preparação para processos seletivos na área de tecnologia.
